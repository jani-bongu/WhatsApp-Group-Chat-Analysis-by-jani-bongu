03/04/24, 8:45 am - Suraj: **Linear Programming (LP)**:

Linear Programming (LP) is a mathematical optimization technique used to find the best outcome in a mathematical model that is represented by linear relationships. It is widely used in economics, engineering, business, and other fields where resources need to be allocated efficiently.

**Assumptions of Linear Programming**:

1. **Proportionality**: Linear Programming assumes that the relationship between variables is linear. This means that changes in decision variables have a consistent and proportional effect on the objective function and constraints.

2. **Additivity**: LP assumes that the overall objective function and constraints are additive, meaning that the total contribution of each decision variable to the objective function or constraint can be obtained by summing the individual contributions of each variable.

3. **Certainty**: Linear Programming assumes that all parameters in the model, such as coefficients in the objective function and constraint equations, are known with certainty and do not vary over time.

4. **Non-negativity**: LP typically assumes that decision variables cannot take negative values. This makes sense in many practical applications, such as production quantities, where negative quantities are not meaningful.

5. **Divisibility**: LP assumes that decision variables can take any real value within a feasible range, including fractional values. This implies that resources can be divided or allocated in any proportion.

6. **Independence**: Linear Programming assumes that decisions made regarding one variable do not affect the feasibility or optimal solution of other variables. In other words, variables are assumed to be independent of each other.

7. **Linearity**: As the name suggests, LP assumes that both the objective function and the constraints are linear. This means that variables are raised to the first power and are not multiplied or divided with each other. If the problem contains nonlinearities, it cannot be solved using linear programming techniques.

8. **Rationality**: Linear Programming assumes that decision-makers are rational and aim to maximize profits, minimize costs, or achieve other objectives in a logical and consistent manner.

These assumptions collectively provide the framework for constructing and solving linear programming models, allowing decision-makers to make optimal decisions in various real-world scenarios. However, it's essential to validate these assumptions in specific applications to ensure the reliability and accuracy of the model's results.

\section{Introduction To Our Project:}
\par This project aims to develop accurate model for bankruptcy prediction using a comprehensive database of Taiwanese companies. Financial statement data for publicly traded Taiwanese firms from 1999-2009 were obtained from the Taiwan Economic Journal database. The data includes financial ratios related to profitability, liquidity, leverage, activity, and growth for both bankrupt and active companies.

\par When a big company goes bankrupt, it can cause serious problems. Being able to accurately predict if a company might go bankrupt is very important for banks, investors, government regulators, and the companies themselves. Spotting money troubles early allows people to take action, make smart decisions, and try to prevent big losses.
\par In the past, prediction bankruptcy mostly relied on analyzing a company's financial numbers and expert opinions. But these methods can be biased or miss important patterns. Thanks to machine learning and advanced math techniques, we can now build better bankruptcy prediction models. These use large data sets and complex patterns to more accurately judge a company's financial health. This study aims to create a powerful bankruptcy prediction model using many different financial and non-financial factors about companies.We'll look at how well it works compared to older techniques and existing models.
\par Having an accurate bankruptcy prediction model is really important for finance, risk management, corporate leadership, and government policies. By giving early warnings, it can help stabilize financial systems, Protect people's interests, and make the overall economy more resilient.
